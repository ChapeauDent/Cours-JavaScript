🚀 Comment visualiser votre GitBook

Voici plusieurs méthodes pour voir votre GitBook en action :

🌐 Méthode 1 : GitBook.com (Recommandée)

Avantages : Rendu parfait, hébergement gratuit, partage facile

Étapes :

  1. Créez un compte sur GitBook.com
  2. Créez un nouveau "Space"
  3. Importez vos fichiers markdown ou synchronisez avec GitHub
  4. GitBook convertira automatiquement vos éléments {% hint %}, {% tabs %}, etc.
📍 URL : https://votre-username.gitbook.io/cours-javascript/

📖 Méthode 2 : GitBook CLI (Alternative)

Avantages : Preview local, pas besoin d'internet

# Installation npm install -g gitbook-cli # Dans votre dossier GitBook gitbook install gitbook serve # Ouverture automatique sur http://localhost:4000

🔧 Méthode 3 : Convertisseur Markdown → HTML

Avantages : Simple, rapide pour un aperçu

# Installation de mdbook (alternative moderne) cargo install mdbook # Ou avec un convertisseur web # Utilisez https://markdowntohtml.com/

💻 Méthode 4 : VS Code avec extension

Avantages : Intégré à votre éditeur

Extensions recommandées :

  • Markdown Preview Enhanced - Aperçu avancé
  • GitBook - Support natif GitBook
  • Markdown All in One - Fonctionnalités complètes
